Mouse and Keyboard

  • q - Quit the application
  • cursor left - previous page(s) (or next page(s) in left to right mode)
  • cursor right - next page(s) (or previous page(s) in left to right mode)
  • cursor up - zoom in
  • cursor down - zoom out
  • plus - if two files are open, next page of the alternate comic
  • minus - if two files are open, previous page of the alternate comic

you can also navigate through pages using the mouse buttons: mouse button 1 to advance, mouse button 3 to go back

If an image is only partially displayed, use mouse drag or mouse wheel to scroll or use

  • 2 - scroll down
  • 4 - scroll left
  • 6 - scroll right
  • 8 - scroll up

Menus

File menu

  • Open - open up to two files. to open a single file, press cancel when prompted for the 2nd file
  • Info - show metadata for the images in all open files

Layout menu

Empty if no files are open

if a single file is open

  • Single Page - display a single page only
  • Two Pages Odd-Even - display two pages, the odd page in first position
  • Two Pages Even-Odd - display two pages, the even page in first position

if two files are open

  • Side by Side - display both files side by side
  • Alternating - display the main file and switch to the alternate file when shift is pressed

Size menu

  • Fit Image - the image is scaled to be as large as possible while still complete visible
  • Fit Width - the image is scaled to be as large as possible while visible in such a way that only vertical scrolling is needed
  • Actual Size - the image will be not scaled

Options menu

  • Show Page Numbers - if enabled, page numbers are shown in the lower left of each panel
  • Right To Left - enables/disables Right To Left mode, if enabled
    • If two files are open the primary file will be on the right
    • If a single files is open with a two-page layout, first position will be on the right
    • wide images will be split right hand half first
    • cursor left will advance pages, cursor right will go backward

Encoding menu

Choose the encoding to interpret image names. Changes will only affect newly opened files

  • Default - use UTF-8 if specified by the zip file, otherwise use the system default encoding
  • 8859-1 - use ISO 8859-1 (Latin 1) encoding
  • CP932 - use CP932 encoding
