
# usage

DuoCBZReader width height path_to_comic1 path_to_comic2

e.g.

DuoCBZReader 1600 1200 comic.en.cbz comic.jp.cbz

# keys

- q: Quit

- cursor left: previous page

- cursor right: next page

- cursor up: zoom in

- cursor up: zoom out

when zoomed in:

- 2: scroll down

- 4: scroll left

- 6: scroll right

- 8: scroll up
